Welcome to ThousandEyes TS Lab

First start by downloading the ThousandEyes Recorder IDE if not already installed.

Once the IDE is installed click on the recorder button and follow the steps given in the question

After finishing the tasks steps mentioned, click on the stop recording button in the IDE and play out the generated script.

The script is likely to fail as the recorder IDE generated script is not optimized. It also doesn't handle try-catch exceptions or iframe switching.

In order to resolve the issue, when the selenium script lands on the page where there appears to be an error, add a sleep timer, switch back to the Recorder IDE and hit the pause button. Doing this would allow you to inspect the chromium webkit instance page to see the different attributes of the element having an issue.

Also add sleep timers appropriately or better use the 'until' utility to wait for elements to load properly before being clicked on to resolve most "Unable to locate element" issues
